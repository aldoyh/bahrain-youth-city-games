# bahrain-youth-city-games

Interactive Kiosk Games Platform - Three games for Bahrain Youth City with real landmark images, bilingual AR/EN support, and RTL layouts

## Overview

This project is an interactive kiosk games platform designed for the Bahrain Youth City Games. It features three engaging games that utilize real landmark images from Bahrain, providing an immersive experience for players. The platform supports both Arabic and English languages, with right-to-left (RTL) layouts for Arabic content to ensure a seamless user experience. The games are designed to be visually appealing and interactive, encouraging players to learn about Bahrain's culture and landmarks while having fun. The platform is built with a focus on accessibility and user engagement, making it suitable for a wide range of audiences.
